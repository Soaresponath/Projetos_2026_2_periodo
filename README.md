# Projetos 2026/2 - 2º Período

Este repositório reúne atividades, exercícios e projetos desenvolvidos durante o 2º período da graduação em Ciência da Computação.

Aqui estão armazenados códigos de estudo relacionados à linguagem C, HTML e fundamentos de programação, com o objetivo de registrar minha evolução acadêmica e prática na área da Computação.

## Sobre o repositório

O objetivo deste repositório é organizar os exercícios desenvolvidos durante as aulas e estudos, envolvendo lógica de programação, estruturas de decisão, operadores, criação de páginas HTML e primeiros projetos práticos.

Os arquivos representam atividades feitas ao longo do processo de aprendizagem, servindo como registro da minha caminhada na programação.

## Conteúdos estudados

Neste repositório, há exercícios envolvendo:

- Linguagem C
- Estrutura básica de programas
- Entrada e saída de dados
- Variáveis
- Operadores aritméticos
- Operadores lógicos
- Estruturas de decisão
- Cálculo de notas
- Condições verdadeiro/falso
- Criação de páginas HTML
- Tabelas em HTML
- Formulários web
- Organização de arquivos no GitHub

## Linguagens utilizadas

## Objetivo de aprendizagem

Este repositório tem como finalidade:

- Praticar programação em C;
- Desenvolver o raciocínio lógico;
- Compreender o uso de operadores e estruturas de decisão;
- Criar páginas simples utilizando HTML;
- Aprender a organizar projetos no GitHub;
- Registrar minha evolução durante o curso de Ciência da Computação.
